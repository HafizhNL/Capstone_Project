Capstone - Project

Proyek ini adalah untuk mengembangkan sebuah aplikasi web 
menggunakan Django dengan konsep CRUD yang memungkinkan pengguna 
untuk berinteraksi dengan data secara efisien. 
Proyek ini bertujuan untuk:

- Menyediakan Antarmuka Pengguna: Memudahkan pengguna dalam mengakses dan mengelola informasi.
- Mengelola Data: Menggunakan database untuk menyimpan dan mengambil data yang diperlukan.
- Menciptakan Fungsionalitas: Menyediakan fitur-fitur yang relevan dengan kebutuhan pengguna atau domain aplikasi.
- Pembelajaran: Menjadi sarana untuk belajar dan meningkatkan keterampilan dalam pengembangan web.
